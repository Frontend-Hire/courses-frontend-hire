# Introduction to React Hooks

This is a fully interactive tutorial that will teach you almost everything you need to know about React Hooks.

We will only cover the most important hooks in this tutorial. Here are the hooks we will cover:

- `useState`
- `useRef`
- `useEffect`
- `Custom Hooks`
- `useContext`
- `useReducer`
- `useCallback`
- `useMemo`
